com.liferay.apio.architect.annotation

Annotation Type Vocabulary.Field


  • @Retention(value=RUNTIME)
 @Target(value=METHOD)
public static @interface Vocabulary.Field
    Defines an annotation that provides information about a field. This annotation should always be used on an interface method.

    This annotation has attributes to customize the schema URL (schema.org by default) and the field description (in the case of a custom field).
