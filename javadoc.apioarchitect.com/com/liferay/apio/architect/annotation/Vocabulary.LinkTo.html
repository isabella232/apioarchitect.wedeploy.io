Annotation Type Vocabulary.LinkTo


  • @Retention(value=RUNTIME)
 @Target(value=METHOD)
public static @interface Vocabulary.LinkTo
    Defines an annotation that indicates a field should be expressed as a link (URI) to another resource. The linked resource class must be provided as the value resource().

    The value resourceType() can be used to differentiate between links to single resources (Vocabulary.LinkTo.ResourceType.SINGLE) and links to collections (Vocabulary.LinkTo.ResourceType.CHILD_COLLECTION).
