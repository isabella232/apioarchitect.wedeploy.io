Interface Form<T>

  • Type Parameters:
    T - the type used to store the Form information

    @ProviderType
public interface Form<T>
    Holds information about an operation's form. The get(Body) method method uses the HTTP request body to extract the form values as detailed in the Form.Builder.

    You should always use a Form.Builder to create instances of this class.
